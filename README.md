# TalentForge LMS

A modern SaaS Learning Management System inspired by TalentLMS.

## Overview

TalentForge LMS is a cloud-based learning platform that enables organizations, instructors, and learners to create, manage, deliver, and track online training programs.

The platform supports course creation, learner management, assessments, certifications, analytics, and multi-organization deployments through a scalable web application.
